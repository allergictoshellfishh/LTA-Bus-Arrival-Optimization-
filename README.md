# LTA-Bus-Arrival-Optimization-
Built a data pipeline using Alteryx Designer to integrate LTA Bus Arrival and Passenger Volume APIs for real-time analytics.

•	Automated data extraction, cleaning, and transformation processes to identify bus congestion patterns at Ngee Ann Polytechnic.
•	Designed Power BI dashboards comparing scheduled versus real-time intervals to recommend peak-hour bus redistribution strategies.
•	Streamlined updates using Power Automate for continuous data visualisation and performance monitoring.
